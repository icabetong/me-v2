# me
A minimalist portfolio website built using React.js with Next with the styling powers of TailwindCSS.

## Built with
* TypeScript
* React.js
* Next.js
* TailwindCSS
* Framer Motion
* and other [cool libraries](https://github.com/icabetong/me/blob/main/package.json)
